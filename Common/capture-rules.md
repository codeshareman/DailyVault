# Capture Rules

## Default routing

| Input | Destination |
| --- | --- |
| 今日行动、判断、待处理 | `Daily/YYYYMMDD.md` |
| Web clipping / article capture | `Clippings/` |
| Bookmark / tool / link | `Resources/` |
| Personal thought / rough research | `Notes/` |
| AI investment report | `Investing/reports/YYYYMMDD-<topic>.md` |
| Publishable candidate | ZNorth Automation queue |
| Source-backed learning candidate | Nervia promotion |

## Daily rule

不要把 Daily 目录变成分类容器。每天默认一个 Markdown 文件。
