# Summaries

从 Daily、Sources、Clippings 和 Notes 汇总出的周/月/季/年回顾。

回顾重点：

- 本周期完成了什么。
- 重要输入和输出是什么。
- 看了、听了、学了、去了、练了什么。
- 哪些兴趣正在增强，哪些值得继续挖掘。
- 健身、路线、身体状态有什么变化。
- 哪些待办迁移、放弃或变成长期项目。
- 哪些内容适合公开到博客或 API。
- 下个周期最重要的目标是什么。

## Dataview-first evidence

回顾页先看统计，再写判断。推荐维度：

- Daily 覆盖：本周期有多少天真实记录。
- Life Timeline / 输入：Daily 保持可读 Markdown 链接；统计主要从链接到的 Sources frontmatter 汇总。
- Sources：按 `category`、`source_type`、`interest_tags`、`visibility` 统计资料池。
- Notes：按 `status` 统计闪念处理状态。
- Tasks：列出 Daily/Notes 未完成事项，辅助迁移或放弃。

AI 只能基于这些可见证据总结，不补写未观察到的完成、情绪、地点或健康数据。
