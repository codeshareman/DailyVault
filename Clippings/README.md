# Clippings

Obsidian Web Clipper 和文章/网页剪藏的默认目录。

规则：

- 剪藏文件由 Obsidian Web Clipper 插件生成，格式以插件配置为准。
- 不维护单独的 DailyVault clipping template。
- 原文和剪藏放这里，不按日期塞进 `Daily/`。
- `Daily/YYYYMMDD.md` 只记录链接、摘要、判断和下一步。
- 如果剪藏需要分类、评分、打 tag、进入年度回顾或博客 API，应在 `Sources/` 建一个 source record，并链接回 clipping。
- 剪藏如果没有后续用途，应在复盘时归档或删除。
